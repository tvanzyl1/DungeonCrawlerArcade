## Concept summary

Create a premium-feeling browser dungeon crawler with:

- playable on PC, tablet, and phone
- desktop controls using **WASD + mouse aim**
- mobile/tablet controls using **twin-stick touch controls**
- automatic slashing or shooting in the direction of aim
- colourful dark-background presentation
- playful arcade feel rather than horror or grimdark tone
- readable HUD and juicy feedback

## Technical preference

- HTML + CSS + JavaScript
- Canvas-based gameplay
- No unnecessary libraries
- Self-contained and easy to host on GitHub Pages

## Theme direction

This is **not** a zombie game. Re-theme toward:

- dungeon halls
- torch-lit chambers
- glowing portals
- slimes / skeletons / bats / enchanted armour / goblins
- treasure drops
- magical weapons and upgrades

Think **arcade fantasy adventure** with humour and punchy feedback.
